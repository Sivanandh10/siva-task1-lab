# Web Server Mission Control

> **What is this lab?**
> A real **nginx** web server is running inside this sandbox. You will use the Terminal to run commands and the **HTML Editor** to browse and edit the actual web files — no browser preview needed, every change you make is live on the server instantly.

---

## How to use this lab

| Tab | What it does |
|-----|-------------|
| **Terminal** | Run Linux commands directly on the web server |
| **HTML Editor** | Browse and view the nginx HTML files in real time |

---

## Your environment

The nginx web server is already running. Confirm it:

    curl localhost

You should see the Mission Control homepage HTML. Your job across the three chapters is to customize it and add a new page.
